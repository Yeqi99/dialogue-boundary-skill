# Evidence Pack

This evidence-pack is for paper coauthors who need to judge what the current repository can actually support.

`paper-pack/` explains how to write the paper. `evidence-pack/` explains what the paper can responsibly claim, based on the current evidence.

Current status:

- This is seed evidence, not final experimental evidence.
- Existing examples and tests are illustrative examples or seed cases.
- The repository does not yet contain real model-output collection, human annotation results, multi-model comparisons, or statistical findings.
- Paper writers should review this folder before drafting claims.
- The paper should not make claims stronger than the available evidence.

Use this folder to decide:

- which claims can be stated as definitions or method proposals
- which claims can be stated as observations from seed cases
- which claims require empirical validation before publication
- which missing evidence should be marked as TODO
