# Paper Pack

This folder is a writing package for developing a paper, technical report, preprint, workshop paper, or short paper about LLM Communication Boundary Skill.

It is designed for a coauthor who did not participate in the earlier project discussion. It contains problem definitions, terminology, taxonomy, method description, dataset design, annotation guidance, experiment planning, metrics, related-work search directions, outline, introduction drafts, and writing constraints.

## Recommended use

1. Read `writing-brief-for-coauthor.md` first.
2. Read `paper-brief.md` for the research summary.
3. Read `problem-definition.md`, `core-concepts.md`, and `taxonomy.md` to stabilize terms.
4. Use `method-boundary-card.md`, `dataset-design.md`, `annotation-guideline.md`, `experiment-plan.md`, and `evaluation-metrics.md` to draft the method and empirical sections.
5. Use `paper-outline.md`, `intro-draft-zh.md`, and `intro-draft-en.md` to start the first paper draft.

## Scope

This material is suitable for:

- a technical report
- a preprint
- a workshop paper
- a short paper
- a research-oriented open-source project document

It does not assume top-conference readiness. If the target is a higher-level archival venue, the project needs stronger experiments, related work, dataset scale, baseline comparisons, and statistical analysis.

## Suggested writing path

Start with a Chinese technical paper draft to stabilize the concept and examples. Then expand it into an English version with a sharper related-work section, formal definitions, evaluation protocol, and limitations.

Do not cite private conversations. Use only anonymized, general, publicly understandable examples.
